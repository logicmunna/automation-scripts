# Selenium Practice Scripts

Python + Selenium দিয়ে লোকেটর (ID, XPath, CSS) এবং ইন্টারঅ্যাকশন প্র্যাকটিস। DemoQA, Saucedemo ইত্যাদি সাইটে ১৫+ স্ক্রিপ্ট।

## স্ট্রাকচার
- scripts/: মেইন স্ক্রিপ্টগুলো
- utils/: কমন ফাংশন
- tests/: টেস্ট কেস (ভবিষ্যতে)

## Setup
1. git clone https://github.com/yourusername/selenium-practice.git
2. cd selenium-practice
3. pip install -r requirements.txt
4. python scripts/demoqa_login.py  # উদাহরণ

প্রথম স্ক্রিপ্ট: DemoQA Login (success case)।
